分享6个超级好用的脚本

今天咱们废话不多说，直接进入主题
推荐几个我认为必备的油猴脚本
PS:因为这些脚本都非常好用，你可能已经在别的UP那边看到过了，或者说你已经装好了

1. AC-百度重定向

是否看腻了百度搜索引擎？在平常我们想用它搜点东西都跳出一堆广告，非常引人生气。

正常搜索问题时右边总是弹出无关的内容

这也导致大部分人都不会选择百度，而是使用bing、google等引擎，但是其中又有问题，因为咱们搜索的东西大部分都是国内的，使用外国引擎就不会把我们真正想要的内容放在最前面
Bing国内版不算，不要扯这个

那么有没有一种方法，能让我们既使用百度搜索，又能免除掉广告呢？

答案是肯定的！

这是安装脚本后的百度搜索

脚本地址：https://greasyfork.org/zh-CN/scripts/14178

脚本功能说明：本脚本功能和标题描述是一致的，就是：
去掉百度、搜狗、谷歌搜索结果的重定向，回归为网站的原始网址
添加百度、搜狗、谷歌搜索结果中Favicon显示效果
搜索高亮用这个：搜索高亮脚本
百度和谷歌搜索页面可以设置为单列、双列模式
添加标记数量，标记当前的id，界面更好看
请求是异步请求，并不会出现一个链接没有反馈回来，其余等待请求是异步请求，并不会出现一个链接没有反馈回来，其余等待的情况，每个链接的请求都是独立的，互不影响，对于网络的影响几乎没有
PS:作者说firefox浏览器可能会不支持，请使用Edge浏览器或Chrome浏览器安装此脚本

2. Github 增强 - 高速下载

受够了从Github上下载的速度慢的可怜？那你就应该用这个！
安装该脚本后，在下载文件时会多出几个选项

图是脚本开源地址截的的

可以选择自己认为距离最近的一个点下载，速度配上IDM还是可以的

地址：https://greasyfork.org/zh-CN/scripts/412245
